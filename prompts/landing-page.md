# Landing Page Prompt

Copy this into Codex after cloning the repo:

```text
Turn this repository into a landing page for [product or business].

Requirements:
- Keep it static HTML, CSS, and vanilla JavaScript only
- Keep GitHub Pages compatibility
- Use a clean modern responsive layout
- No external framework dependency

Please:
- Create a strong hero with a clear value proposition
- Add benefits, proof, CTA, FAQ, and contact sections
- Keep the copy concise and conversion-focused
- Add one primary CTA throughout the page
- Update the page title, meta description, and social copy
- Keep the code readable and simple
```
