# First Codex Prompt

Use this as your first prompt after cloning:

```text
Turn this repo into a personal website for [name], a [role] based in [location].

Constraints:
- Keep it compatible with GitHub Pages
- Keep it as static HTML, CSS, and JavaScript
- No framework, no build step
- Keep the content on one page unless a second page is clearly useful

Make these changes:
- Rewrite all placeholder copy
- Add a more specific hero section
- Add 3 projects with believable summaries
- Update the contact section
- Improve the color palette and typography for [industry/style]
- Tighten mobile spacing

Also update the README so the next person understands what changed.
```
