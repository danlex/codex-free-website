# Astro Website Starter

This starter uses Astro and is configured for a static GitHub Pages deployment.

## Before you publish

1. Update `site` and `base` in `astro.config.mjs`
2. Rename `public/CNAME.example` to `public/CNAME` if you use a custom domain
3. Push to GitHub and enable Pages with GitHub Actions

## Local development

```bash
npm install
npm run dev
```
