# Next.js Website Starter

This starter uses Next.js static export so it can be deployed to GitHub Pages.

## Constraints

GitHub Pages only serves static files. Keep the site within what Next.js static export supports.

## Local development

```bash
npm install
npm run dev
```

## Publish it

1. Rename `public/CNAME.example` to `public/CNAME` if you use a custom domain
2. Push to GitHub
3. Enable Pages with GitHub Actions
