# LinkedIn Post

I put together a small starter repo for anyone who wants a personal website without paying for hosting or learning a heavy stack first.

It uses:

- Codex to edit the site
- GitHub for version control
- GitHub Pages for free hosting
- Your own domain for the final polish

The repo stays intentionally simple: static HTML, CSS, and JavaScript. Clone it, open it in Codex, rewrite the content, push to GitHub, enable Pages, connect your domain, and you are live.

What is inside:

- a one-page website starter
- a quick setup guide
- a starter Codex prompt
- LinkedIn launch assets you can reuse

If you have been putting off making a site because it felt like too much setup, this removes most of the friction.
