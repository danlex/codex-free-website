# AGENTS.md

## Repo purpose
This repository is a template-first launch kit for building a free website with AI using Codex and GitHub Pages.

## Product promise
Use this template. Edit with Codex. Preview locally. Publish on GitHub Pages. Add your own domain.

## Rules
- Keep the project static: HTML, CSS, and vanilla JavaScript only.
- Do not add frameworks, bundlers, package managers, or build tooling unless explicitly requested.
- Optimize for GitHub Pages compatibility first.
- Keep "Use This Template" as the dominant CTA.
- Treat custom domain setup as part of the standard onboarding flow.
- Prefer simple, readable code and concise, clear copy.
- Avoid duplicate explanations across multiple files.
- The site should feel like a reusable product, not a personal portfolio starter.
